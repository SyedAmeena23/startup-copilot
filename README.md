# 🚀 Startup Copilot

AI-powered multi-agent startup planning platform.

## Features

- AI Startup Planner
- Market Analysis
- Competitor Analysis
- Financial Planning
- Product Strategy
- Marketing Strategy
- Startup Score
- Firebase Authentication
- PDF Export
- Startup History
- Dashboard Analytics

## Tech Stack

Frontend
- React
- Tailwind CSS
- Recharts

Backend
- FastAPI
- LangGraph
- Gemini API

Database
- SQLite

Authentication
- Firebase

PDF
- ReportLab

## Installation

Frontend

```bash
npm install
npm run dev
```

Backend

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
